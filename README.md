# Projecto final En Dos Ruedas

BackEnd por la application de fin de curso.
La aplicacion esta montada en NESTjs.
Para usarla clonar el repositorio y ejecutar los comando descripton abajo.

## Installation

```shell
npm install
```

## Running the app

```shell
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## Test

```shell
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```
